== Description ==

[با همکاری سایت نخبر] (http://nakhabar.com)
Nakhabar News scroll farsi news text fetch from nakhabar.com; Widget is a plugin to automatically scroll up/down the content inserted in the description +/- hashtags and short news.
You can add your news, updates and ads into description box to be shown via between news.

* Show scrolling text fetch from nakhabar.com
* Can be used as advertising, notice, important updates
* Compatibility with many other plugins
* Option to scroll text up or down
* Option to increase or decrease speed of the scroll

The source of information is provided by [nakhabar.com] (http://nakhabar.com)
Follow nakhabar on Twitter to keep up with the latest updates [Nakhabar](http://twitter.com/nakhabar/)

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory
the top-level folder. Just upload all the files and put them in '/wp-content/plugins/'.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit your Widgets Section (*NakhabarNews*)
4. Drag and widget in the sidebar.
5. Enter title, description and checkbox options that you need.  News and/or Hashtags text will automaticall fetch from nakhabar.com, title will not scroll.
6. Configure any options as desired then enable the plugin.
7. That's it!

If you have to upgrade manually simply repeat the installation steps and re-enable the plugin.


== Frequently Asked Questions ==

= Can I change the direction of scroll? =

Yes, you can change the direction of scrolling text up or down from the plugin setting options given in the admin section of wordpress (Manage Nakhabar News).

= Can I change speed of scroll? =

Yes, you can change the speed of scroll from the plugin setting options given in the admin section of wordpress (Manage Nakhabar News).
